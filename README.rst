Introduction
============

``ProofReader`` is a command line interface to the Netspeak web
service http://www.netspeak.org/

The main utility is a command line program called ``proofread``.


Install
=======

::
   pip install ProofReader


Usage
=====

From a terminal, type

::
   $ proofread "query"
